# 🌃NO.99

> 탈출한 실험체와 도시의 빼앗긴 빛을 다룬 분위기 중심 네온 액션 사이드 스크롤러

---

## 🎮게임 소개

**NO.99**는 정부에 의해 통제된 황폐한 도시를 배경으로 한 2D 사이드스크롤 액션 게임입니다. 

붕괴된 도시는 정부의 로봇과 드론이 순찰하는 폐쇄 구역이 되었습니다.  

플레이어는 **Pulse**를 흡수해야만 생존할 수 있는 불안정한 실험체를 조작하며, 자신을 추적하는 정부 소속 로봇과 드론을 피해 도시를 빠져나가야 합니다. 

**Pulse**는 생존, 전투, 환경 조작에 사용되는 핵심 자원입니다. 

플레이어는 **Pulse**를 흡수하고 활용하며 각 구역을 돌파하고, 자신이 왜 추적당하는지 점차 알아가게 됩니다.

### 핵심 기능

1. **Pulse 흡수** — Pulse를 흡수해 생존 에너지를 유지합니다.  
2. **Pulse 조작** — Pulse를 사용해 적을 무력화하고, 장치와 환경을 조작하며 구역을 돌파합니다.  
3. **도시 탈출** — 정부의 추적을 피해 도시 밖으로 나아갑니다.

## 💡중요 팁

- 첫 번째 방에서는 블라인드를 열어야 다음 구역으로 이동할 수 있습니다
- 오브젝트 위에 마우스를 올리면 상호작용 키가 표시됩니다
- 전투 중 대시(`SHIFT`)를 적극적으로 사용하는 것을 추천합니다
- `Q` 스킬을 자주 사용하세요

# ⚙️설정

`ESC` 키를 눌러 설정 메뉴를 열 수 있습니다.

### 설정 가능 항목

- 화면 크기 변경
- VSync ON/OFF
- 볼륨 조절
- 게임 종료

### 메뉴 조작

| 키     | 행동    |
| ----- | ----- |
| W / S | 메뉴 이동 |
| ENTER | 적용    |

### 전체화면

`ALT + ENTER`
