게임 방법

- 게임을 실행하기 위해 EXE 파일을 누르세요
- ENTER 키를 눌러 게임을 시작하세요
- 대화창을 따라가며 게임을 플레이하세요
  !!!!!! 중요 !!!!!!
  - 첫번째 방에서, 다음 스테이지로 넘어가기 위해 블라인드를 올려야 합니다
  - 오브젝트에 마우스 커서를 올리면 마우스 커서가 바뀝니다
  - 팁) 대쉬와 쉬프트를 활용하면 유리합니다

메인 키

- 'A' - 왼쪽으로 움직임
- 'D' - 오른쪽으로 움직임
- 'SPACE' - 점프
- 'S' - 움크리기
- RIGHT CLICK - 펄스 흡수
- LEFT CLICK - 적 공격
- SHIFT - 대쉬

세팅

- ESC 키를 누르면 세팅창이 열립니다

- 세팅창에서 W와 S로 위와 아래로 움직일 수 있습니다

- 아래 가능
  
        - 전체화면/창모드
	- 프레임 변경 가능(30/60/144/240/제한 없음)
        - VSYNC 키고 끄기 - VSYNC를 키면 노트북의 최대 주사율로 맞춰집니다
        - 세팅창에서 EXIT로 게임을 종료할 수 있습니다
        - 볼륨 조절 가능

- ALT + ENTER 누르면 전체 화면 전환
