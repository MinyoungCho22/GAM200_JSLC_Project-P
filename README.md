NO.99   
is an atmospheric, action side-scroller where you control an escaped 
experiment that can absorb and gear up the city’s neon energy, ”Pulse”.
With the pulse will you share this stolen light to become a savior to 
its citizens, or consume it all to become the perfect tool you were designed to be?        

Repo : https://github.com/MinyoungCho22/GAM200_JSLC_Project-P.git

Software/hardware requirements: 
Visual Studio is Required 
Windows 10 and up is Required

Installation Instructions:
  - Unzip the downloaded file 
  - Look for this file: The Solution File ending in .sln.
  - What to do: Double-click the .sln file. This opens the entire "Solution,"
    which includes all your source code, headers, and project settings (libraries, include directories).

    
Look at the toolbar at the top. You will see two dropdown menus.
Configuration:
Debug: Use this while writing code. It turns off optimizations so you can step through code line-by-line, but it runs slower.
x64: Standard for almost all modern computers.

How to complie:
Build Only: Ctrl + Shift + B (Checks for errors without running).
Run (with Debugging): F5 (Press this to test your code).
Run (No Debugging): Ctrl + F5 (Faster start, keeps the console window open after the program finishes).

PLAY:
Black Screen with glitching title will appear PRESS ENTER to play the game

Objective: Survive and escape the city!

SETTING - 
  - PRESS ESC KEY TO OPEN THE SETTING
  - IN THE SETTING PRESS W OR S TO GO UP AND DOWN AND PRESS ENTER TO APPLY IT
  - YOU CAN CHANGE SCREEN SIZE OR TO EXIT THE GAME IN THE SETTING
  - YOU CAN PRESS ALT + ENTER FROM FULL SCREEN TO TAB SCREEN
    
MAIN KEY
  - 'A' - MOVE LEFT
  - 'D' - MOVE RIGHT
  - 'I' - ABSORB PURSE
  - 'J' - FUNCTION KEY(OPEN DOOR, OPEN WINDOW, ETC)
  - 'S' - CROUCH, HIDE, ETC
  - 'SPACE' - JUMP
