HOW TO PLAY
  - PRESS THE EXE FILE TO OPEN THE GAME
  - WHEN YOU OPEN THE FILE PRESS ENTER TO START THE GAME
  - FOLLOW THE TEXT ON THE BOTTOM TO PLAY THE GAME

MAIN KEY
  - 'A' - MOVE LEFT
  - 'D' - MOVE RIGHT
  - 'E' - ABSORB PURSE
  - 'F' - FUNCTION KEY(OPEN DOOR, OPEN WINDOW, ETC)
  - 'S' - CROUCH, HIDE, ETC
  - 'SPACE' - JUMP
